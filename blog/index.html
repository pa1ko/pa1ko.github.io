      F-string

      Patryk Konarzewski

      2019-11-308

      Nowa funkcjonalność dodana w Pythonie 3.6, f-string jest nowym sposobem interpolacji stringów. Używa się go przez dodanie f'' przed stringiem.

      In [14]:
      name = 'Yukka'
age = 56

print('Hello {name}, age: {age}')
print(f'Hello {name}, age: {age}')

      Hello {name}, age: {age}
Hello Yukko, age: 56

      Wewnątrz f-string'a można umieszczać wyrażenia, które będą ewaluowane w czasie wywyływania kod.

      In [9]:
      f'{13 * 5}'
      Out[9]:
      '65'
      In [16]:
      def reverse(text):
    return ''.join(reversed(text))

f'{reverse(name)} olleH'
      Out[16]:
      'okkuY olleH'
      In [22]:
      text = (
    f'Hello {name} '
    f'your age is: {age}'
)

text
      Out[22]:
      'Hello Yukko your age is: 56'

      Dla zwykłego grupowania wystarczy skorzystać z klasy Grouper

          ['one', 'two', 'one', 'two', 'three', 'one']]

cols = pd.MultiIndex.from_tuples(tuples, names=['first', 'second'])

index = pd.MultiIndex.from_tuples([('a', 'a'), ('a', 'b'), ('b', 'a')], names=['foo', 'bar'])

df = pd.DataFrame(np.random.randn(3, 6), index=index, columns=cols)
