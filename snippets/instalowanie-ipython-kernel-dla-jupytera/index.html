      Instalowanie nowych kerneli dla Jupytera

      Patryk Konarzewski

      2019-10-292

      Instalowanie IPyhon kernel dla Jupytera:

      1. source activate .env
      2. pip install ipykernel
      3. python -m ipykernel install --user --name {{nazwa_kernela}} --display-name "{{Ładna nazwa}}"
